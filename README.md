# [DEPRECATED] VisTrails Provenance Software Development Kit

**This project is not actively maintained.**

This package contains the source code for the VisTrails Provenance
Software Development Kit (ProvSDK) as well as some examples on its
use. It was developed commercially by VisTrails, Inc. in 2009-2011,
but is now available as open-source.

The main idea of the SDK is to enable software developers to capture
and utilize provenance information in their work.  The ProvSDK
contains infrastructure for creating, searching, and replaying
provenance information; developers need only define serializations for
the changes from state to state.  The examples are the best
reference for the code.
